# Austrian-Smartmeter-Tool
Tool with Raspberry and Interface to bring Data to MQTT


# INTERFACE
## ENGLISH

The interface is a simple level converter. The M-BUS operates up to 48V. The internal serial interface is used as the input.

COMPONENTS:

-> 1x 4N25 diode

-> 1x 1.5 kΩ resistor

-> 1x 4.7 kΩ resistor

-> 1x 24V Zener diode
